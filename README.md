# Heart Disease Prediction — EDA + 3 ML Models

End-to-end ML project predicting heart disease using UCI Cleveland dataset.

## Results
| Model | Accuracy |
|-------|----------|
| 🏆 Logistic Regression | 91.67% |
| Random Forest | 88.33% |
| Gradient Boosting | 86.67% |

## Dataset
- 297 patients, 13 clinical features
- Source: UCI Heart Disease Cleveland Dataset

## What I Did
- Exploratory Data Analysis (EDA)
- Feature correlation heatmap
- Age & cholesterol distribution analysis
- Trained & compared 3 ML models
- Feature importance analysis

## Top Predictors
1. thalach — Max Heart Rate (0.159)
2. cp — Chest Pain Type (0.123)
3. oldpeak — ST Depression (0.115)

## Tools Used
Python · Pandas · NumPy · Scikit-learn · Matplotlib · Seaborn
